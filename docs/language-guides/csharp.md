# C# / .NET Guide

- **Framework:** .NET 8; nullable reference types enabled.
- **Patterns:** DI via built-in container; Options pattern for config; records for value objects.
- **Async:** Use async/await end-to-end; avoid sync-over-async.
- **Testing:** xUnit/NUnit, Moq; Coverlet for coverage; 90%+ targets.
- **Analyzers:** Enable Roslyn analyzers; StyleCop rulesets.
