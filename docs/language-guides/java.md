# Java Guide

- **Framework:** Spring Boot 3.x; records for DTOs; `jakarta` annotations.
- **Tooling:** Maven/Gradle; SpotBugs, Checkstyle, Error Prone.
- **Patterns:** Hexagonal Architecture; interfaces for ports; Spring beans for adapters.
- **Testing:** JUnit 5, Mockito, Testcontainers; PIT mutation testing preferred.
- **Null Safety:** Use `Optional` thoughtfully; avoid `null` in domain.
